Проект представляет собой веб-сайт, посвященный кибербезопасности. 
Его цель заключается в демонстрации важности кибербезопасности и предоставлении доступной и 
понятной информации для широкой аудитории. Это позволит повысить осведомленность 
о цифровых угрозах и методах защиты от них.

Используемые технологии:
1. HTML - Основной язык разметки, используемый для создания структуры веб-страницы. 
2. CSS - Стилизация элементов страницы осуществляется через внешний файл style.css, который определяет визуальное оформление приложения.
3. Google Fonts - Подключение шрифта Jura через API Google Fonts для улучшения визуального восприятия текста на странице.
4. Использование изображений в качестве фона и иконки к сайту, структурирование контента с помощью списков (<ul> и <li>), 
 использование гиперс ссылок <a>.
5. Сайт опубликован на GitHub Pages (GitHub — это облачная платформа для хостинга IT-проектов и совместной разработки,
под капотом которой находится популярная система контроля версий Git)

Файлы проекта:
index.html - главная страница (Типы угроз кибербезопасности)
rules.html - страница о правилах обеспечения кибербезопасности
style.css - стилизация сайта
readme.txt - описание проекта
public/bg.avif - фон для сайта
public/favicon.png - иконка для сайта
